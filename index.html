Discuss the scope of var, let, and const

1. var has function scope and is hoisted, allowing access before declaration.
2. let and const have block scope, restricting their accessibility to the block where they are defined.
3. const enforces immutability after initialization, while let allows variable reassignment.

Tell us the use cases of null and undefined

null is typically used to intentionally indicate the absence of a value or an empty object reference, while undefined often represents uninitialized variables or missing object properties in JavaScript.

What do you mean by REST API?

A REST API is a set of rules and conventions for building and interacting with web services using HTTP methods to perform CRUD operations on resources represented as URLs.
